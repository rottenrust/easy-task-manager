# Personal Task Manager Backlog

## Цель проекта

Создать локальный персональный таск-менеджер для повседневного использования с мобильного устройства через Telegram-бота.

На первом этапе приложение должно помогать быстро фиксировать бытовые дела, идеи и привычки, а также вручную получать свежий список активных задач по запросу.

## Требования

### Входит в ближайший этап

- Мобильный интерфейс через Telegram-бота.
- Локальное хранение данных.
- Быстрое добавление задач из Telegram.
- Просмотр активных задач.
- Отметка задач выполненными.
- Автоматический перенос выполненных задач в архив.
- Просмотр архива.
- Доступ только для владельца бота.
- Простые команды и понятные сообщения.
- Архитектура, готовая к будущему расширению.

### Пока не входит

- Дедлайны.
- Приоритеты пользовательских задач.
- Проекты и теги.
- Повторяющиеся задачи.
- Импорт и экспорт.
- Web UI.
- Синхронизация между устройствами.
- Автоматические напоминания.

## Релизы

### Release 0.1: Ядро задач

Цель: подготовить приложение к развитию и отделить бизнес-логику от интерфейсов.

Результат: приложение умеет работать с активными и архивными задачами на уровне ядра.

### Release 0.2: Telegram MVP

Цель: дать первый мобильный интерфейс для ежедневного использования.

Результат: пользователь может вести личный список дел с телефона через Telegram.

### Release 0.3: Удобство повседневного использования

Цель: сделать Telegram UX менее техническим и более приятным.

Результат: бот становится удобным для регулярного использования.

### Release 0.4: Стабилизация

Цель: повысить надежность локального продукта.

Результат: приложению можно доверять личный список дел.
